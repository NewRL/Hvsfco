HMS Online Appointment Booking


This Module will Add functionality to Allow patients to Book an Appointment online from the portal

Create appointment form on the portal.

Detailed Description of Module

Online Appointment Booking By ACS

The patient can create an appointment by selecting a date and available slot.

After selecting a proper date and slot patient dan do payment and complete booking.

Admin can create Slots manually and automatically.

Limit of possible booking per slot can be configured.

Slot creation can be done easily using a wizard.

The online booked appointment can be seen in a separate menu and can be managed easily.

Admin can configure advance booking day availability.
